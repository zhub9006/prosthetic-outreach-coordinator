# Contributing to Prosthetic Outreach Coordinator

Thank you for your interest in contributing to our prosthetic outreach initiative! This guide will help you get started.

## 🤝 How to Contribute

### 1. Explore Issues
- Visit our [Issues tab](https://github.com/zhub9006/prosthetic-outreach-coordinator/issues) to find tasks.
- Each issue includes a region, task type, and acceptance criteria.
- Filter by labels to find tasks matching your skills and region.

### 2. Claim a Task
- Comment on an issue to claim it (e.g., "I'll take this on this week!").
- Only one person claims an issue at a time — please communicate if you need to hand it off.

### 3. Work & Document
- Complete the task according to the acceptance criteria.
- Push your work to a branch and link it back to the issue.
- For clinical research tasks, update the `CLINICAL_TRIALS_REFERENCE.md` with new findings.

### 4. Submit a Pull Request
- Open a PR with a clear description of your changes.
- Reference the issue number in your PR description (e.g., "Closes #1").
- Ensure all checklists are marked complete.

## 📋 Issue Templates

We use structured templates to keep tasks consistent:

- **Outreach Task Template** — for regional coordination work
- **Clinical Research Template** — for trial analysis
- **Logistics & Supply Chain Template** — for sourcing & coordination
- **Community Outreach Template** — for events & partnerships

## 🚀 Getting Started

1. Clone the repository: `git clone https://github.com/zhub9006/prosthetic-outreach-coordinator.git`
2. Create a branch: `git checkout -b my-task`
3. Make your changes
4. Commit and push
5. Open a Pull Request

## 💬 Questions?

Open a discussion or comment on an existing issue. We're here to help!
