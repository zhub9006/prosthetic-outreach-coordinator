# Open Issue Templates Checklist

Before creating any issue, please ensure all required fields are filled in.

## Standard Issue Checklist
- [ ] Region is specified (`region:chicago`, `region:boston`, `region:atlanta`)
- [ ] Type is specified (`type:research`, `type:logistics`, `type:community`, `type:clinical`)
- [ ] Acceptance criteria are defined with measurable checkboxes
- [ ] Description includes context and background
- [ ] Related NCT IDs (if clinical trial) are linked
